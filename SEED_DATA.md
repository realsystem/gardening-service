## Growing Requirements

### Sun Requirements
- **Full Sun** - 6-8 hours daily (Most vegetables, fruits, grains)
- **Partial Sun** - 4-6 hours daily (Lettuce, spinach, arugula, some herbs)
- **Partial Shade** - 2-4 hours daily (Mint, lemon balm, woodland plants)

### Water Requirements
- **High** - Lettuce, spinach, cucumbers, celery, microgreens
- **Medium** - Most vegetables, herbs, and fruits
- **Low** - Mature perennial herbs, grains, cover crops

## Database Schema

```sql
CREATE TABLE plant_varieties (
    id SERIAL PRIMARY KEY,
    common_name VARCHAR(100) NOT NULL,
    scientific_name VARCHAR(200),
    variety_name VARCHAR(100),
    days_to_germination_min INT,
    days_to_germination_max INT,
    days_to_harvest INT,
    spacing_inches DECIMAL(5,2),
    row_spacing_inches DECIMAL(5,2),
    sun_requirement sunrequirement,
    water_requirement waterrequirement,
    description TEXT,
    growing_notes TEXT,
    photo_url VARCHAR(500),
    tags TEXT  -- Comma-separated string: 'vegetable,warm-season,fruit'
);
```

### Verify Data
```sql
-- Count varieties
SELECT COUNT(*) FROM plant_varieties;  -- Should be 107 (expanded) or 27 (basic)

-- Group by primary category
SELECT
    SPLIT_PART(tags, ',', 1) as category,
    COUNT(*) as count
FROM plant_varieties
WHERE tags IS NOT NULL
GROUP BY category
ORDER BY count DESC;

-- View all vegetables
SELECT common_name, variety_name, days_to_harvest
FROM plant_varieties
WHERE tags LIKE '%vegetable%'
ORDER BY common_name, variety_name;

-- View fastest to harvest
SELECT common_name, variety_name, days_to_harvest
FROM plant_varieties
WHERE days_to_harvest IS NOT NULL
ORDER BY days_to_harvest
LIMIT 20;
```

## Using Plant Varieties in the App

### Frontend Flow
1. **Create a Garden** - Select garden type (outdoor/indoor/hydroponic)
2. **Add Seed Batch** - Choose from 107 plant varieties (alphabetically sorted)
3. **Add Planting Event** - Select plant variety and garden location
4. **View Details** - See germination time, harvest days, care requirements
5. **Track Growth** - Log irrigation, soil samples, and observations
6. **Get Tasks** - Auto-generated care tasks based on plant needs

### API Endpoints
```bash
# Get all plant varieties (returns 107 varieties, sorted alphabetically)
GET /plant-varieties

# Search by common name
GET /plant-varieties?search=tomato  # Returns 6 tomato varieties

# Filter by tags (currently returns all matching)
GET /plant-varieties?search=microgreen  # Returns 5 microgreen varieties
```

### UI Features
- **Alphabetical Sorting**: All dropdowns show varieties sorted by common name, then variety name
- **Search & Filter**: Type to search in CreateSeedBatch component
- **Count Display**: Shows total varieties available (e.g., "107 available")
- **Clear Labeling**: Common Name (Variety Name) format for easy selection
